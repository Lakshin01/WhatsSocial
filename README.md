## ğŸš€ Getting Started

### Prerequisites

- Bun Package Manager
- Docker
- Kubernetes

### Local Development

1. **Clone the repository**

    ```
    git clone https://github.com/Lakshin01/WhatsSocial.git
    ```

2. **Navigate to the project directory and install dependencies using Bun**

    ```
    cd WhatsSocial
    bun install
    ```

3. **Run the application locally**

    ```
    bun start
    ```
